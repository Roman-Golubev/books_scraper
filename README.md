# Проект books_scraper

## Цель  
Целью проекта является сбор данных о книгах, размещённых на сайте <https://books.toscrape.com/>,  
и представление этих данных в генерируемом файле books_data.txt.  

## Инструкция по запуску
**1 Скачать проект по ссылке <https://github.com/Roman-Golubev/books_scraper/tree/main>.**  
**2 В терминале, находясь в каталоге проекта, выполнить команду: `pip install -r requirements.txt`.**  
**3 Открыть ноубук "HW_03_Р.О.Голубев.ipynb" (далее - Ноутбук), находящийся в папке "notebooks".**  
**4 Выполнить импорт библиотек в поле в верхней части Ноутбука (команда - `Run this cell and advance`).**   
**5 В разделе "Задание 2. Сбор данных обо всех книгах" последовательно:**      
*5.1 запустить выполнение верхнего поля;*  
*5.2 проверить, что в нижнем поле в функции `scrape_books` переменной `is_save` передаётся значение `True`;*  
*5.3 запустить выполнение нижнего поля.*  
**6 Файл "books_data.txt" будет сгенерирован в папке "artifacts".**  
**7 Для тестирования работы основных функций проекта в разделе Ноутбука  
"Задание 4. Написание автотестов" запустить выполнение поля.**
